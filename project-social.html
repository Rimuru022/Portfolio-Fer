Community Data Dashboard

Built a dashboard for a local NGO to track community health metrics and visualize impact over time.

The Challenge

The NGO had been collecting community health data for years but had no effective way to analyze or present it. Data lived in scattered spreadsheets.

Before state: scattered spreadsheets

They needed a centralized dashboard that could aggregate data from multiple sources, provide visual insights, and help them tell compelling stories to donors and stakeholders.

What I Built

I developed an end-to-end data solution for the organization:

  • Data pipeline to clean and consolidate information from multiple sources
  • Interactive dashboard with filters for time periods, demographics, and program areas
  • Automated reporting that generates monthly impact summaries
  • Visual charts showing trends in health outcomes, attendance, and program reach
  • Export functionality for presentations and grant applications

Results

The dashboard transformed how the NGO operates. They can now identify at-risk communities in real time, measure program effectiveness with concrete metrics, and present clear evidence of impact to donors. Monthly report generation that used to take 3 days now happens automatically.

Dashboard view with charts and metrics

Tools & Technologies

  • Python (pandas for data cleaning)
  • SQL for data queries and aggregation
  • BI Dashboard tools
  • Data visualization libraries
  • Automated scheduling with cron jobs
